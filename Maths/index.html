BIDMAS Rounding Percentages

Math Lessons - BIDMAS

By the Moon Walkers



Learning Goal: To understand and be able to use BIDMAS to solve problems.

Definition: B.I.D.M.A.S is the order in which an equation is completed.

Equations are completed left to right, with the higher priority being completed first. For example, in 2 x 4 + 3, 2 would be multiplied by 4, and then 3 would be added. Brackets group together numbers. For example, in 2(4 + 3), 2 would be multiplied by 4 + 3, which is 7. So this would be 2 x 7.

2 x (4 + 3)

First, the equation in the brackets must be figured out, as the first letter of Bidmas, B, stands for brackets. This equates to 7. The equation then becomes 2 x 7, which is 14.

2 x (4 + 3)

= 2 x 7

= 14

17 + 7 x (0 x 13)

First, the equation in the brackets must be figured out, as the first letter in BIDMAS. B stands for brackets. 0 X 13 equates to 0. The equation then becomes 17 + 7 x 0. Multiplication is the next step in BIDMAS. 7 x 0 equals 0. The equation then becomes 17 + 0, which is 17.

17 + 7 x (0 x 13)

= 17 + 7 x 0

= 17 + 0

= 17

7 + 4 + (6 x 9)

First, the equation in the brackets must be completed, as it is the first letter in BIDMAS

7 + 4 + (6 x 9)

= 7 + 4 + 54

= 11 + 54

= 65

89 + (6 - -3 [5 + 6] + 25)

First, the brackets are solved. Because there is a bracket inside another bracket, the innermost bracket is solved first. There is also a number next to the bracket, without any math symbol next to it. This means that whatever the bracket equates to is multiplied by that number. In this case, 5 + 6 = 11, and then 11 is multiplied by the number next to it, which is negative three. This makes -33. The equation is now 89 + (6 - -33 + 25). When there are two negative symbols right next to each other, it is the same as a +. So 6 - -33 is equal to 6 + 33. This allows us to solve the brackets. 6 + 33 is 39, and 39 + 25 is 64. All that’s left is 89 + 64, which is 153.

89 + (6 - -3 [5 + 6] + 25)

= 89 + (6 - -3 x 11 + 25)

= 89 + (6 - -33 + 25)

= 89 + (39 + 25)

= 89 + 64

= 153

9 + 7 + (3 x 20) ÷ 10

B being the first letter of BIDMAS, the brackets are figured out first. 3 x 20 is 60. The new equation is 9 + 7 + 60 / 10. Next is multiplication. Because there are no other multiplication signs in the new equation, we can move onto division. 60 / 10 is 6. The new equation is 9 + 7 + 6. Using simple addition, we can determine that the answer is 22.

9 + 7 + (3 x 20) / 10

= 9 + 7 + 60 / 10

= 9 + 7 + 6

= 16 + 6

= 22
